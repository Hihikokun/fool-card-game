This is a recreation of the popular Eastern European game "Durak" (Fool).


Rules

End each turn with the "End Turn" button in the top left corner of the screen. If the bot isn't playing anything, then the bot has no more valid plays left, and you should end the turn for them.


Thank you for checking out this project :). I might rework it in the future to be more optimized, improve the AI, and remove the plethora of bugs.
